# emotion-kwon
## Personal sentiment analysis combining objective data and subjective feelings
结合客观数据和主观感受的个人情感分析：

中心思想：主客观数据整合能使情绪判断更精准。
情感是瞬时动态平衡模型，每个人的情绪总量恒定。但是在不同事件下不同感受分配到的情感指数有不同。
每个人的感受程度会因为环境变化，状态等改变，并且不同人的情感体验也不同，例如高敏感人群对情感的体验会更高。

目前可以画出不同主观情绪配比，一周的压力值变化，主客观融合的压力值（目前采用简单加权平均）（客观压力值需要自行使用智能手表提供的值）
